版本2（建议使用）

  • 展示方式：有盖楼
  • SVN:https://svn.intra.sina.com.cn/ui/project/new_comment
  • Demo:https://svn.intra.sina.com.cn/ui/project/new_comment/html/带评论框列表.html

    <link rel="stylesheet" type="text/css" href="http://ent.sina.com.cn/470/2014/0324/comment.2.min.css">
<div class="sina-comment-wrap" id="SI_FormList1"></div>
<script src="http://i.sso.sina.com.cn/js/ssologin.js" charset="utf-8"></script>
<script src="http://ent.sina.com.cn/470/2014/0324/comment.2.min.js"></script>
<script>
    ;(function(exports){
        var $ = exports.___sinacMNT___;
        var FormList11 = new $.cmnt.FormList('SI_FormList1',{
            // 频道
            channel:'v4test',
            // 新闻id
            newsid:'3-1-3',
            parent:'',
            // 编码，默认为gbk,可使用utf-8,一般和页面编码一致
            encoding:'gbk'

            // 频道
            channel: 'v4test',
            // 新闻id
            newsid:'3-1-3',
            // 评论类型，单条新闻为0（如正文页评论），多条新闻为1（如高清图评论）

            encoding:'gbk',

            page: 1,
            // 每次显示条数，默认为20

            hideList:0,

            // 最新评论默认显示条数

            clickMoreTimes:3,
            // 加载完成回调，可在回调中获取评论数等评论数据
            loaded:function(self){
                // console.log(self);
            }
        },{});
    })(window);
</script>

版本3

  • 展现方式：无盖楼，原盖楼形式转为微博转发的形式（//@XXX:...）
  • SVN:https://svn.intra.sina.com.cn/ui/project/comment3
  • Demo:https://svn.intra.sina.com.cn/ui/project/comment3/html/main.html
  • 使用方法： 
    <link rel="stylesheet" type="text/css" href="http://news.sina.com.cn/css/87/20140926/comment.3.css">
<div class="sina-comment-wrap" id="SI_FormList1"></div>
<script src="http://i.sso.sina.com.cn/js/outlogin_layer.js" charset="utf-8"></script>
<script src="http://news.sina.com.cn/js/87/20140926/comment.3.min.js"></script>
<script>
    ;(function(exports){
        var $ = exports.___sinacMNT___;
        // 保持和页面编码一致，gbk或utf-8
        $.cmnt.config.encoding = 'utf-8';
        var FormList1 = new $.cmnt.FormList('SI_FormList1',{
            // 频道编号
            channel:'yl',
            // 新闻id
            newsid:'28-3-4188538',
            // 默认为空
            parent:''
        },{
            channel: 'yl',
            // 新闻id
            newsid:'28-3-4188538',
            // 默认为0，多条新闻（图片）集合的评论为1
            group: 0,
            // 开始页数，默认为1
            page: 1,
            // 每页评论条数
            pageSize: 20,
            // 最大楼层
            maxFloor:1,
             // 隐藏评论列表，只显示评论框
            hideList:0
            // // 最新评论默认显示条数
            // hotPageNum:5,
            // // 最新评论默认显示条数
            // firstPageNum:5,
            // // 可点击最新评论正文“更多评论”按钮次数,达到该次数后，显示“查看更多评论”链接，再点击则跳转
            // clickMoreTimes:3
        },{});
    })(window);
</script>

注意

  1. 为了便于跟踪维护，请不要使用版本2，版本3之外的相关css及js。
  2. 版本2,目前使用范围最广。
  3. 版本3,目前只用14年版正文，模仿微博转发的形式，展示上有很多问题，如大段文字（如上1000字）无法区分用户名及回复关系，版本2较之，盖楼更清晰。
  4. 评论不要放置在过小的容器内，既会导致布局错乱，还会影响用户使用体验。
  5. 因为评论有审核机制，评论失功后（前端已显示），刷新页面后评论可能不显示，亦可能不显示，点“先赞”功能类似。
  6. 最热评论是否，及显示条数，完全由接口控制。
  7. 接口问题请咨询王磊wanglei12，如评论接口，获取列表接口及点赞接口等。
